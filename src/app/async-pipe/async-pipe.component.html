Async Pipe need to SUBSCRIBE A Variable

Getter with delay operator => {{ (Data1 | async) || 'no-data' }}

Getter without delay operator => {{ (Data2 | async) || 'no-data' }}

A Property assigned string in Observer with delay operator => {{ Data3 || 'no-data' }}

A Property assigned Observable<string> with delay operator => {{ (Data4 | async) || 'no-data' }}

A Function returns Observable<string> with delay & concatMap operator => {{ (getData5() | async) || 'no-data' }}

You need a single Observable instance. The async pipe keeps a reference to the observable and subscribes to it. When you use the getData function call, a new observable is created each time there's a change detection, and so the async pipe subscribes to the new observable and doesn't wait for the previous one anymore.
